## 🚀 Getting Started

To run the project locally:

1. Clone the repository:

git clone https://github.com/Sanjanayadav07/My-Portfolio.git

2. Navigate to the project directory:

cd My-Portfolio

3. Install dependencies:

npm install

4. Start the development server:

npm run dev
